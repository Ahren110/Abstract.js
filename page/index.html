Simple RenderModel

RenderModel is Using for Rendering Page Blocking;Here is a sample for RenderModel; 
    var header = new RenderModel({
      el: ".header",
      url: "/cgi/headerData",
      processData: function(data){
      },

      template: "<li soda-repeat='item in list'>{{item.name}}</li>"
    });

    header.rock();

Simple ScrollModel

ScrollModel is Using for Rendering Blockings which loads more content when users pull down list to the bottom. 
    var header = new ScrollModel({
      el: ".list",
      scrollEl: ".list",
      url: "/cgi/headerData",
      processData: function(data){
      },

      template: "<li soda-repeat='item in list'>{{item.name}}</li>"
    });

    header.rock();

MultitabModel

MultitabModel is one of Relationship Model, which tells the relationship between models; 
      var tab1 = new RenderModel();
      var tab2 = new RenderModel();

      var tab = new MultitabModel();

      tab.add(".tab1", tab1);
      tab.add(".tab2", tab2);

      tab.rock();
